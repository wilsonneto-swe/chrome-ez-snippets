Privacy Policy

Ez Snippets — Chrome Extension · Last updated: February 22, 2025

Ez Snippets does not collect, transmit, or share any user data. Everything stays on your device.

What data is stored

The extension stores only the snippets you create (title and content). This data is saved in your browser's local storage and never leaves your device.

Clipboard access

When you click the copy button on a snippet, the extension writes that snippet's content to your system clipboard using the browser's Clipboard API. No clipboard data is read or stored by the extension.

Language detection

The extension reads your browser's language setting (navigator.language) to display the interface in English or Portuguese. This value is not stored or transmitted anywhere.

Permissions used

  • storage — allows the extension to persist your snippets in the browser's local storage.
  • sidePanel — allows the extension to open as a Chrome side panel.

Data collection

Ez Snippets does not:

  • Collect personal information
  • Use analytics or tracking
  • Make network requests
  • Send data to external servers
  • Use cookies
  • Access browsing history or any other browser data

Third-party services

The extension does not integrate with any third-party services or APIs.

Data deletion

You can delete individual snippets at any time from within the extension. To remove all data, uninstall the extension or clear your browser's local storage for the extension.

Changes to this policy

If this policy is updated, the changes will be reflected on this page with an updated date.

Contact

If you have questions about this privacy policy, please open an issue on the project's GitHub repository.
